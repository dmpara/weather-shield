# weather-shield

This project is to make a self-contained/self-sustained weather station built around the particle photon.

Current Hardware:
- photon - https://www.particle.io
- SparkFun Photon Weather Shield (DEV-13630) - https://www.sparkfun.com/products/13630
- Voltaic 3.5 Watt Solar Charger Kit - http://www.voltaicsystems.com/3-5-watt-kit
- SparkFun 2.4GHz Duck Antenna RP-SMA (WRL-00145) - https://www.sparkfun.com/products/145
- SparkFun Interface Cable RP-SMA to U.FL (WRL-00662) - https://www.sparkfun.com/products/662

Future Hardware:
- MOD-1016 AS3935 lightning and storm sensor - http://www.embeddedadventures.com/as3935_lightning_sensor_module_mod-1016.html
- SparkFun Weather Meters (SEN-08942) - https://www.sparkfun.com/products/8942
